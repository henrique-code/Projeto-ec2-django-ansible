# Provisionamento de Instância EC2 com Terraform (Alura)

Este projeto automatiza a criação de uma infraestrutura básica na AWS utilizando Terraform. O objetivo principal é subir uma instância EC2 de forma rápida e programática.

## 🚀 Tecnologias Utilizadas
* [Terraform](https://terraform.io)
* [AWS CLI](https://amazon.com)
* [Provedor AWS](https://terraform.io)

## 📋 Pré-requisitos
Antes de começar, você vai precisar:
1. Ter o **Terraform** instalado.
2. Ter o **AWS CLI** configurado com suas credenciais (`aws configure`).
3. Uma conta ativa na AWS.
